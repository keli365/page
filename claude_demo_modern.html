🤖 System Prompt（系统核心）

来源：Anthropic 内置

是否可修改：❌ 不可修改

说明：定义 Claude 的基础能力边界、安全准则和默认行为模式

🧠 Memory（记忆库）

路径：~/.claude/memory/

是否可修改：✅ 可编辑

配置方式：

  • 对话中说 "记住我是C++老师"
  • 直接编辑 ~/.claude/memory/MEMORY.md

📋 CLAUDE.md（项目档案）

路径：./CLAUDE.md（当前工作目录）

是否可修改：✅ 可编辑

配置方式：在项目根目录创建 CLAUDE.md 文件，定义项目特定的指令和规范

📜 Rules（动态规则）

路径：.claude/rules/ 或子目录中的 CLAUDE.md

是否可修改：✅ 可编辑

作用：按目录层级定义规则，覆盖或补充上级配置

使用场景：

  • 代码规范：src/ 目录使用特定命名法
  • 测试要求：tests/ 目录必须包含单元测试
  • 文档标准：docs/ 目录使用特定模板

示例：创建 src/.claude/rules.md

## 代码规范
- 使用蛇形命名法（snake_case）
- 所有函数必须添加中文注释
- 头文件必须使用 #pragma once

⚡ Skills（技能系统）

路径：settings.json / settings.local.json

是否可修改：✅ 可编辑

配置方式：

{
  "skills": {
    "generate-exercise": {
      "description": "生成练习题",
      "prompt": "..."
    }
  }
}

🔌 MCP（外部工具连接）

路径：settings.json

是否可修改：✅ 可编辑

配置方式：

{
  "mcpServers": {
    "sqlite": {
      "command": "uvx",
      "args": ["mcp-server-sqlite", "--db-path", "..."]
    }
  }
}
